Proxymeter
==========

Proximity App based on Estimote beacons.

Requirements
------------

* Xcode 8.3
* Cocoapods
* Estimote beacons

Building
--------

```
$ cd ios/Proxymeter
$ pod install
```

Open `Proxymeter.xcworkspace` and run project in real device (iOS 10+). Enter and exit from beacon region and expect local notifications in foreground & background.
